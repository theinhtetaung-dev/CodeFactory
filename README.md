# 🏭 CodeFactory

Welcome to **CodeFactory**! This repository serves as my personal playground and testing environment for exploring bleeding-edge technologies, frameworks, and concepts that I haven't worked with yet. 

## 🚀 What's Inside?

This repo is a collection of isolated testing projects and proof-of-concepts (PoCs). Whenever I encounter a new technology, I build a small project here to understand how it works under the hood.
